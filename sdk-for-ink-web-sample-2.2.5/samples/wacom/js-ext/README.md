# js-ext

## Description

The Javascript extension. Provides additional functionality to classes when they are available and when particular functionality is missing.

 * Object
 * String
 * Number
 * Array
 * ArrayBuffer
 * TypedArray
 * Screen
 * HTMLElement
 * HTMLImageElement
 * Image
 * Canvas (Adds support for OffscreenCanvas when is not available)
 * DOMPoint
 * DOMRect
 * DOMMatrix
